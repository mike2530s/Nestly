Nestly

Digital Platform for Property Rental Management

Instituto Tecnológico Sanmiguelense • 2025

We present a comprehensive digital platform that democratizes access to advanced technological tools in the real estate sector. Nestly eliminates technological barriers enabling small real estate agencies to compete with high-cost solutions through modern technologies like Angular, Laravel, and Docker.

Real Estate Technology Gap

Real estate agencies, especially smaller ones, face significant barriers from existing high-cost platforms, confusing interfaces, and rigid functionality that limits their digital competitiveness.

Critical Problems Identified:

  • • High costs of current commercial systems
  • • Non-intuitive and complex user interfaces
  • • Restricted functionality without local adaptability
  • • Lack of flexibility for specific business needs

Modern Technology Architecture

Nestly implements a Single Page Application (SPA) with decoupled architecture, where Angular manages dynamic interfaces communicating with a Laravel RESTful API for optimized data operations.

Modern Frontend:

  • • Angular for dynamic and responsive interfaces
  • • PrimeNG for advanced UI components
  • • Responsive design for multi-device support
  • • Optimized user experience architecture

Robust Backend:

  • • Laravel Framework for secure RESTful API
  • • Efficient database management systems
  • • Docker for portability and deployment
  • • Simplified scalability and maintenance
